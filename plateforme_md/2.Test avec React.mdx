Les tests avec React c'est comme les tests JavaScript, juste qu'ici il va nous falloir
un utilitaire pour vérifier dans le HTML généré par React. On va donc utiliser
[testing-library](https://testing-library.com/docs/react-testing-library/intro) qui
va nous permettre de tester le composant comme un utilisateur le verrait.

Mais avant d'utiliser testing-library, on va essayer de faire sans, pour bien comprendre
l'utilité d'une tels librairie.
